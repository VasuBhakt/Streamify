# Streamify Frontend

A modern, responsive React-based frontend for the Streamify video streaming platform. Built with Vite, Redux Toolkit, and TailwindCSS, and hosted on Vercel.

> **Note:** This is proprietary software and is not open source. All rights reserved.

- **Electric Blue Theme**: Modern aesthetics with Dark Mode support

### üé¨ Video Experience
- **Custom Player**: Playback controls, volume, and progress seeking
- **Lazy Loading**: Optimized thumbnail delivery via Cloudinary

- **Auth System**: Persistent sessions with automatic token refresh

- **Interactions**: Like videos, comment system, and channel subscriptions
